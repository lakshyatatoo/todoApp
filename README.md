# Todo App

A simple, interactive **Todo App** built with **HTML, CSS, and JavaScript**.  
Track tasks, mark them completed, delete them, and persist them using **localStorage**.

---

## Features

- Add tasks with a **title** and **due date**
- Delete individual tasks
- Clear all tasks at once
- Responsive layout with vibrant colors
- Data persists after page reload

---

## Usage

Open `index.html` in your browser.  
Add tasks, delete or clear them, and stay organized!  
Perfect for **personal productivity**.

---

## Files

- `index.html` – App structure  
- `style.css` – Styling and responsive design  
- `script.js` – App logic (add/delete/render tasks, localStorage)
